< : less than (보다 작다) == 오른쪽으로 열린 부등호
> : greater than (보다 크다) == 왼쪽으로 열린 부등호
& : ampersand == 일반적인 and 의 줄임 기호
© : copylight == 동그라미에 c 가 들어가있는 기호
® : 등록된 마크 == 동그라미에 R이 들어가있는 기호
≤ : less than equal (보다 작거나 같다) == 오른쪽으로 열린 부등호 아래 등호 기호
≥ : greater than equal (보다 크거나 같다) == 왼쪽으로 열린 부등호 아래 등호 기호
  : 공백(space bar) 1칸을 나타내는 엔티티
