in-line css

css 는 세 가지 형태로 HTML 문서에 삽입할 수 있다.

  • in-line
  • internal css
  • external css

in-line css : 는 HTML 의 시작 태그에 style="" 속성으로 명시

in-line 방식으로는 문서 전체에 걸친 스타일 적용 불가. 스타일이 지정된 해당 태그 1개에만 적용.
